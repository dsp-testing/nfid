# NFID ICP Wallet Kit

This repository contains the NFID ICP Wallet Kit, which is a set of libraries and tools for integrating ICP Wallets into decentralised applications on the Internet Computer.

## Packages

The following packages are included in this repository:

- [`@nfid/icp-wallet-kit`](./icp-wallet-kit.mdx) - A library to integrate compatible ICP Wallets into your application.
- [`@nfid/icp-wallet-kit/react`](./icp-wallet-kit-react.mdx) - A library containing react specific components for integrating compatible ICP Wallets into your application.
